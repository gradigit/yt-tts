What it is

Text in, voices out

Like Bumblebee speaking through radio clips -- yt-tts searches millions of YouTube transcripts, finds exactly where your words are spoken, extracts the audio with 30ms precision, and stitches it all together.

No API keys. No cloud services. No synthetic voices. Every word comes from a real human who recorded themselves on YouTube. The result is imperfect, textured, and alive.

For agents

Give your agent a voice

The installer sets up a skill for Claude Code, Codex CLI, and any agent that reads from standard skill directories. Agents call yt-tts with --json for structured output.

Agent invocation
# Agents call yt-tts with --json
yt-tts --json "hello world"
Structured output
{
  "output_path": "yt-tts-a1b2c3d4.mp3",
  "duration_ms": 3200,
  "clips": [
    {
      "video_id": "dQw4w9WgXcQ",
      "phrase": "hello world",
      "confidence": 0.84
    }
  ],
  "missing_words": [],
  "exit_code": 0
}

Install

One command

Installs everything: uv, ffmpeg, yt-dlp, the CLI, agent skills, and a starter transcript index.

curl -LsSf https://raw.githubusercontent.com/gradigit/yt-tts/master/install.sh | sh

Then:

yt-tts "the meaning of life"
